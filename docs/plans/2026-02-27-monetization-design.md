# Monetization Strategy Design

**Date**: 2026-02-27
**Topic**: Hybrid Freemium Economy & Monetization

## 1. Core Economy & "Opals" Ecosystem
*   **Primary Currency**: Zen Gems (Opals).
*   **The "Energy" Mechanic**: Opals act as the energy required to start any given mindfulness/puzzle session within the app.
*   **Acquisition**: 
    - Base Daily Login Rewards.
    - Rewarded Video Ads (User-initiated, clear value exchange).
    - Social Media Sharing.
    - Referrals.

## 2. Dynamic Scalable Pricing
The Opal cost to initiate a session is not static; it scales based on the complexity and value of the experience requested by the user:
*   **Grid Complexity**: Higher number of elements/shapes increases the cost.
*   **Time Duration**: Longer sessions cost exponentially more Opals.
*   **Premium Audio**: Selecting top-tier, binaural, or exclusive soundscapes adds a premium cost.

## 3. Multiplayer Mode Multiplier
*   **Solo Mode**: Operates at the 1x Base Cost multiplier.
*   **Cooperative Mode**: Operates at a significantly higher multiplier, reflecting the added value and server load of the synchronized zen experience.
*   **The "Social Surcharge"**: If a Premium user hosts a Cooperative room with maxed-out settings and premium audio, any Free user joining that room *must* still pay the equivalently high Opal "entry fee" for those specific settings. This exposes the Free user to the premium experience while draining their Opals, driving conversion.

## 4. The "Friction vs. Freedom" Upsell
Rather than establishing a hard paywall when a user depletes their Opals, the app will present a fork in the road, giving the user agency:
*   **Option A ["The Friction"]**: Watch a 30-second ad to earn a flat rate of Opals (e.g., 50 Opals) to continue playing.
*   **Option B ["The Freedom"]**: Start a 5-Day Free Trial of the Premium Tier.
*   **Psychological Engine**: Once a user is tired of managing the Opal economy and experiences the absolute freedom of the 5-day trial, the churn rate back to the free tier is statistically minimized.

## 5. The Premium Tier
*   **Limitless Tranquility**: Premium users completely bypass the Opal economy. 
*   **Features**: They can play infinitely with maxed complexity grids, infinite time duration, and full access to the entire premium soundscape and cosmetic (3D crystals) library. No ads are ever served.
